##  What is Kali Linux?
Kali Linux is a Linux distribution designed for penetration testing and security auditing. It comes pre-installed with many tools used for ethical hacking and cyber security analysis.


##  Basic Linux Commands
- ls → Lists files and directories  
- cd → Changes the current directory  
- pwd → Shows the current directory path  


##  Networking Basics
- **IP Address**: A unique identifier assigned to devices on a network  
- **DNS (Domain Name System)**: Translates domain names into IP addresses  


##  Tools (Beginner Level)
- **Wireshark**: A network protocol analyzer used to monitor traffic  
- **Nmap**: A network scanning tool used to discover hosts and services
